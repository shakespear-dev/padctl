# Phase 6: Test Plan — Install CLI, XDG Paths, InputPlumber Devices, COPR, Config UX

Each test maps to a success criterion in `brief.md`.

## Unit Tests

- [ ] TP1: `padctl install --destdir /tmp/test-root --prefix /usr` produces correct directory
  structure: binary, service, udev rules, and devices/ all present under destdir; no
  `systemctl`/`udevadm` invoked when `DESTDIR` is non-empty.
  validates: T1 success criterion

- [ ] TP2: udev rules generated by install contain one `SUBSYSTEM=="hidraw"` rule per unique
  VID/PID across all installed TOML files; no duplicate entries.
  validates: T1/D2 (ADR-008)

- [ ] TP3: `src/config/paths.zig` — `userConfigDir` returns `$XDG_CONFIG_HOME/padctl` when
  `XDG_CONFIG_HOME` is set; falls back to `~/.config/padctl` when unset.
  validates: T4 success criterion

- [ ] TP4: `searchDeviceConfigs` merges three layers with user > system > builtin priority;
  user-layer file shadows builtin file of same name.
  validates: T4 success criterion

- [ ] TP5: Gyro curve — `curve=2.0`, `deadzone=1000`, `GYRO_MAX=32767`: output matches vader5
  reference within ±1 LSB across mid-range input values.
  validates: T11 success criterion

- [ ] TP6: Gyro curve — `curve=1.0`: output identical to prior linear path (normalization is
  transparent for linear case).
  validates: T11 success criterion

- [ ] TP7: Gyro curve — `GYRO_MAX == deadzone`: returns 0 without division-by-zero error.
  validates: T11 guard condition

- [ ] TP8: `StickProcessor.processScrollMode` with `axis_x` pushed full-left and full-right emits
  `AuxEvent{ .scroll_h }` events with correct sign; no `REL_HWHEEL` emitted when axis is in
  deadzone.
  validates: T10 success criterion

- [ ] TP9: `padctl scan --config-dir /tmp/test-root/usr/share/padctl` with mocked VID/PID matching
  Legion Go S (`0x1a86:0xe310`) returns correct TOML path and exits 0.
  validates: T2 + T13 CI step

- [ ] TP10: `padctl scan` with no `/dev/hidraw*` present prints "No HID devices found" and exits 0.
  validates: T2 edge case

- [ ] TP11: `padctl reload` with no PID discoverable exits 1 and prints "padctl daemon not
  running".
  validates: T3 success criterion
